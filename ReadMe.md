## ✨ 주요 기능

- 🔐 회원가입 / 로그인 (JWT 인증)
- 💑 커플 연결 기능
- 📅 일정 관리 (캘린더)
- 📸 추억 기록 (이미지 업로드)
- ❤️ 디데이 관리
- 🔔 알림 기능

---

## 🏗️ 아키텍처

- Frontend: Next.js (CSR + SSR 혼합)
- Backend: Spring Boot REST API
- Database: PostgreSQL
- 인증: JWT (Access / Refresh Token)
- 상태관리: Zustand + React Query
